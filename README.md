# Discord Bot
A Discord Bot made using Python and Discordpy as a pet project.

### External APIs used -
* IMDBpy
* Safygiphy
* Wikipedia
* NewsAPI
* Googletrans
* Gsearch

### Features - 
* Simple games
* GIF search feature
* Daily news feature
* Wikipedia search feature <br>
  Usage: [WIKI!][query]<br>
    Returns the result for the query on wikipedia<br>
* Basic server management commands<br>
  Usage: [INFO!]<br>
    Returns Server name, Member count, Time of Origin, and Owner of the server<br>
  Usage: [ROLES!]<br>
    Returns all the different roles<br>
  Usage: [STATUS!]<br>
    Returns a bar plot of the different statuses of server members<br>
  Usage: [KICK!][user_id]<br>
    Kicks the specified user<br>
  Usage: [BAN!][user_id]<br>
    Ban's the specified user<br>
* Random jokes command
* Language Translation command - translate languages from one to the other.
* Calculator
* Google Search ( Programming Doubts )
